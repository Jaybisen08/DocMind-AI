# 📄 DocMind AI

> **Transform Your Documents into an Intelligent AI Assistant**

DocMind AI is an AI-powered document intelligence platform that enables users to interact with PDFs, DOCX, and TXT files using natural language. Powered by **Google Gemini**, it helps users chat with documents, generate summaries, extract insights, and create quizzes, making learning, research, and document analysis faster and smarter.

Built for **Hacks Day Bhopal 2026**.

---

## 🚀 Problem Statement

**AI Knowledge Retrieval from Personal Files**

**How can users instantly find answers from thousands of their own PDFs, notes, and documents using AI?**

DocMind AI addresses this challenge by transforming static documents into an interactive AI-powered knowledge assistant.

---

## ✨ Features

- 📂 Upload PDF, DOCX, and TXT documents
- 💬 AI-powered chatbot for document Q&A
- 📑 Generate detailed AI summaries

- 📚 Generate quizzes from uploaded documents

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Jaybisen08/DocMind-AI.git
cd DocMind-AI
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the application

```bash
npm run dev
```
