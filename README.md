# 📄 DocMind AI

> **AI-Powered Document Intelligence Platform using Google Gemini**

DocMind AI is an AI-powered web application that helps users interact with their documents using natural language. Instead of manually reading lengthy PDFs, users can upload documents, ask questions, generate summaries, create quizzes, and extract meaningful insights—all powered by Google Gemini.

Built for **Hacks Day Bhopal 2026**.

---

## 🚀 Problem Statement

**Problem Statement #5 – AI Knowledge Retrieval from Personal Files**

> **How can users instantly find answers from thousands of their own PDFs, notes, and documents using AI?**

DocMind AI transforms static documents into an intelligent knowledge assistant, enabling users to retrieve information conversationally instead of manually searching through pages.

---

## ✨ Features

- 📂 Upload PDF, DOCX, and TXT files
- 💬 AI-powered document chat
- 📑 Generate detailed or concise summaries
- 🧠 Extract key concepts and important topics
- 📚 Generate quizzes from uploaded documents
- 📥 Download AI-generated summary as PDF
- 🔍 Smart document search

## ⚙️ How It Works

1. Upload a PDF, DOCX, or TXT file.
2. The document is processed and understood using Google Gemini.
3. Ask questions about the uploaded document.
4. Generate AI summaries.
5. Extract important concepts and insights.
6. Create quizzes from document content.
7. Download the AI-generated summary as a PDF.
